# Gym Pro - Transforme seu corpo

O **Gym Pro** é uma landing page moderna e responsiva para uma academia de alta performance. O projeto foi desenvolvido com foco em oferecer uma experiência de usuário limpa, destacando os planos de treinamento, coleções de vestuário e informações sobre a unidade.

##  Tecnologias

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5**: Estrutura semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
    - **Variáveis CSS**: Para uma manutenção centralizada de cores, fontes e espaçamentos.
    - **Arquitetura de Componentes**: Estilos organizados por componentes (Header, Hero, Plans, etc.) para melhor escalabilidade.
    - **Flexbox e Grid**: Para layouts responsivos e alinhamentos precisos.
    - **Design Responsivo**: Adaptado para diferentes tamanhos de tela (Mobile e Desktop).

##  Funcionalidades

- **Menu Responsivo**: Navegação adaptável com menu hambúrguer para dispositivos móveis.
- **Seção Hero**: Chamada impactante para novos alunos com botões de ação clara (Matrícula e Aula Grátis).
- **Categorias em Destaque**: Galeria visual para diferentes coleções (Casual, Esporte e Moderno).
- **Cards de Planos**: Comparativo visual de planos (Mensal e Anual Pro) com destaque para o plano "Mais Popular".
- **Newsletter**: Área no rodapé para captura de leads e fidelização de clientes.
- **Rodapé Completo**: Links de navegação, redes sociais e informações de suporte.

##  Estrutura do Projeto

```text
gym-pro/
├── css/
│   ├── components/       # Estilos específicos de cada seção
│   ├── base.css          # Estilos globais e tipografia
│   ├── reset.css         # Reset de estilos do navegador
│   └── variables.css     # Definição de cores e variáveis globais
├── images/
│   ├── banners/          # Imagens de fundo e banners
│   ├── highlight/        # Imagens das coleções em destaque
│   ├── icons/            # Ícones utilizados na interface
│   └── logo/             # Logotipos da marca
└── index.html            # Arquivo principal do projeto
```

##  Como executar o projeto

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para uma melhor experiência de desenvolvimento, utilize a extensão **Live Server** no VS Code.
